LINGUAGENS:

html

HTML (Hypertext Markup Language) é a linguagem fundamental para criar e estruturar páginas web,
utilizando uma série de tags para definir elementos como textos, imagens e links.

css

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar e
formatar a apresentação de páginas web criadas com HTML.

JavaScript

JavaScript é uma linguagem de programação essencial para tornar páginas web interativas e dinâmicas.
Ele permite adicionar funcionalidades como animações, manipulação de dados e resposta a ações do usuário em tempo real.

python

Python é uma linguagem de programação de alto nível,
conhecida por sua sintaxe clara e legível, o que facilita a escrita e manutenção de código. Ela suporta múltiplos paradigmas, como programação orientada a objetos, imperativa e funcional.

Projetos:

Conversor de moedas

A função principal de um conversor de moedas é permitir a troca de
valores entre diferentes moedas, com base nas taxas de câmbio vigentes.

Conversor de temperatura

A função de um conversor de temperatura é facilitar a conversão de valores entre diferentes escalas de
temperatura, como Celsius (°C), Fahrenheit (°F), Kelvin (K), entre outras.

Clculadora IMC

A função de uma calculadora de IMC (Índice de Massa Corporal) é determinar
se o peso de uma pessoa está dentro de uma faixa saudável, com base na sua altura e peso.

Urna eletrônica

A função de uma urna pode variar dependendo do contexto, mas, em termos gerais,
uma urna é um recipiente utilizado para guardar, coletar, proteger ou registrar algo.

Lista rápida

A função de uma lista rápida de maneira geral, ela se refere a uma lista ou recurso que tem como
objetivo organizar e acessar informações de maneira eficiente e prática

Card

cards (ou cartões) são usados como unidades de informação
visual compacta, interativa e de fácil acesso.

FAQ

A função de um site com perguntas e respostas é fornecer uma plataforma
onde os usuários possam postar dúvidas e obter
respostas de outros usuários, especialistas ou da comunidade em geral

Refound

A função de um app de reembolso é facilitar o processo de solicitação, aprovação e pagamento de reembolsos por
despesas que devem ser reembolsadas

Galáxias

A função de uma página sobre galáxias é fornecer informações sobre a estrutura,
características e tipos de galáxias, como as espirais, elípticas e irregulares.
